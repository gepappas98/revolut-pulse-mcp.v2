mcprice — Real-Time Market Data for AI Assistants

finance stocks crypto MCP Server No API Key

Get live stock and crypto prices directly inside Claude or Cursor. No API keys. No setup headaches.

⭐ Star on GitHub 🚀 Deploy on Railway 📖 Read Docs

🧰 6 Tools

  • get_price — Current price + 24h change for any stock or ETF
  • get_prices_bulk — Fetch up to 20 tickers at once
  • get_crypto_price — Binance real-time crypto data (BTC, ETH, SOL...)
  • price_snapshot — Full watchlist snapshot: stocks + crypto + market mood
  • revolut_price_check — Price + "is this stock available on Revolut?"
  • crypto_top_movers — Top gainers & losers from Binance 24h

⚡ Why mcprice?

Unlike traditional APIs, mcprice is built specifically for AI assistants. No API keys, no complex setup — just real‑time prices in your conversation. Perfect for portfolio tracking, trading journals, and AI‑powered analysis.

🚀 Quick Start

Install in 2 minutes:

git clone https://github.com/gepappas98/revolut-pulse-mcp.v2.git
cd revolut-pulse-mcp.v2
uv run --with fastmcp,httpx python app.py

🤖 Claude Desktop Setup

Add to your claude_desktop_config.json:

{
  "mcpServers": {
    "mcprice": {
      "command": "uv",
      "args": ["run", "--with", "fastmcp,httpx", "python", "app.py"],
      "cwd": "/path/to/revolut-pulse-mcp.v2"
    }
  }
}

📊 Data Sources & Reliability

  • Stocks & ETFs: Yahoo Finance (no key, 15‑20 min delay on free tier)
  • Crypto: Binance Public API (real‑time, no delay)
  • Caching: 30s for stocks, 10s for crypto
  • Auto‑retry with exponential backoff
